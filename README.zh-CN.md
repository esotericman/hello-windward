# Windward入门级示例

## 通过该项目你将学会以下知识

- 简单的路由用法
- 打包可执行的Jar
- 访问自己的HTML或者其它静态资源

## 该项目运行后会有以下路径可供访问

| 实际的资源                                         | 请求路径                             |
|-----------------------------------------------|----------------------------------|
| 根路经                                           | http://127.0.0.1:8080            |
| 获取JSON形式的返回值                                  | http://127.0.0.1:8080/json       |
| 获取index.html                                  | http://127.0.0.1:8080/index.html |
| 获取inner.html(当然，你也可以尝试直接请求inner.html 看看能不能访问) | http://127.0.0.1:8080/inner      |
